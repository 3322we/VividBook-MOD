# VividBook-MOD
LaTeX 模板 VividBook 的一个改版. 主要特点: 样式多, 可自由定制; 稳健性好, 报错少; 多语言支持.

## 模板发展历史
作者自 2024 年第一次接触 LaTeX 就开始使用 Elegantbook 模板 (以下简称原版). 2025 年 9 月从某在线编辑器的模板库中发现了 Elegantbook 的一个改版 VividBook. 后者更精致漂亮, 但其作者表明了模板盒子太多, 编译速度太慢的问题. 从 2025 年 11 月开始, 作者断断续续地做了如下工作:

- 调整了 VividBook 文档类中的代码;
- 将一些功能如图表, 列表等移植到 Macros/ 中的自定义宏包里, 方便自行选用;
- 删去了不少盒子;
- 修改了一些样式, 如目录和定理盒子颜色等;
- 调整了其他几种颜色主题;
- 适配了中英法三种语言 (对法语的支持略有瑕疵但不碍事);
- 自定义了一些功能如纸张颜色, 章标题样式等.
- ...

这些工作在 2026 年 2 月 6 日宣告正式完成, 并于次日完成 Github 项目的上传. 本模板不敢保证编译速度变快, 但运行还算完美. 而且, 通过**文档类选项**, 模板的样式可深度定制.

## 版本更新历史

2026-02-09 **v1.1**: Others/ 文件夹汇总了三个语言版本的文档以供参考. 删除了 Econfig 包, 将有关代码整合进了 config.tex; 弃用了 Emath 包, 但考虑到不同的需求, 此包被保留到了 Others/ 文件夹中.

## 使用注意事项
本模板的编译环境为 Windows 11 + TeXLive 2025 + VScode LaTeX Workshop 2025 + LaTexmk XeLaTeX, 其他环境理论上可用, 但不清楚是否可能存在字体问题. 不论是用哪种语言, 都请尽量用 XeLaTeX 编译. 虽然文档类中有 IfTeX 命令作为稳健性处理, 但由于未作测试, 不清楚是否有问题. 而且, 其他编译器下的字体支持也不如 XeLaTeX 好. 本模板的编译速度相比 VividBooK 可能有所提升, 但作者未做准确比较.

项目的主文件为 book.tex. 遵循 VSCode 的默认习惯, 编译产生的文件, 包括 PDF 文档和日志文件都在 out/ 中. 文件管理相对严格, 请使用者不要随意折腾文件夹或更改文件的位置, 否则容易导致编译器找不到文件. 

**第一次编译时**, 请将字体选项设为默认值 chinesefont=ctexfont 以及 font=cm (或者直接注释掉), 这是为了防止使用者的编译环境缺少某些字体而导致的编译报错.

中文高亮请使用 config.tex 定义的 \emph 命令, 而不是 \textbf. 因为在使用楷体的环境中, 比如定义环境中, 由于中文***粗斜体***在 ctex 默认字体 fandol 中不存在, 导致 \textbf 可能无法生效, 而且会产生字体警告. (文档第二章的一个定义环境就用了 \textbf, 不过这是照搬 VividBooK 版的一个示例, 不要紧, 改掉就好)

模板代码为同时适配中西文而引入大量 ifdefstring 结构的代码, 使得当语言选项 lang 切换到某语言时, 各标题, 编号等也自动切换到该语言, 并且确保了中文字符不会流入到西文模式中. 西文模式下未引入 ctex 包, 因此, 写西文文档, 请将主文件 (book.tex), 导言区配置文件 (config.tex) 和各章节子文件 (Cha/cha0x.tex) 中的中文字符尽数删除. 为避免西文章标题过长而溢出章头盒子, 模板提供了 chapterhead 选项, 此选项在 off 模式下变为无章头图片的样式, 请参考 frenchversion 文档. 这虽然是一个备用模式, 但也还算美观.

另外, 模板并不能保证其他语言的可扩展性. 如要引入其他语言, 中文文档 out/book.pdf 中有简单的方法说明. 使用者也可参考原版文档类文件. 

本模板提供了五个自定义宏包, 具体功能可参考中文文档. 它们在 config.tex 中得到声明. 其中, Econfig 包是必需的, 其它四种是可选的. 但作者编写说明文档时, 这五个宏包都用了, 使用者在写自己的文档时要注意.

## 作者推荐样式
原 Elegantbook 的文档类选项就已经足够丰富了, 作者此改版又进一步调整和扩充了一些, 使用者可自由定制自己喜欢的样式, 具体设置方法可参考中文文档. 但作者个人最中意的两种样式是:

- 中文模式 (lang=cn), 方正字体 (chinesefont=founder), 白纸 (paper=white, 默认), 蓝色主题 (color=blue, 默认), 有章头图 (chapterhead=on, 默认).
- 西文模式 (lang=en 或 lang=fr), garamond 字体 (font=garamond, 当然 stix2 也不错), 米色纸 (paper=beige), 青色主题 (color=cyan), 无章头图 (chapterhead=off). 可参考 frenchversion 文档.
